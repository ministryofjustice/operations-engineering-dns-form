Getting Started

This service is used for requesting DNS changes. Before you begin, please read the following information carefully to understand the process and the current constraints.

What you need to know

  • You will need either a @justice.gov.uk or @digital.justice.gov.uk email in order to authenticate on the next page.
  • All DNS requests must comply with our DNS standards and guidelines.
  • If you are creating or modifying an existing record, please ensure you know the DNS record name and details of the new record value.
  • If you are requesting a different type of DNS change, for example registering a new domain, please ensure you include as much information about the request as possible in the provided text field.
